# 画像・スクリーンショットの取り扱い（日本語）

日本語 | [简体中文](README.md)

 - 画像は `docs/images/` に配置します
- README のヒーロー画像は `docs/images/hero.png`（差し替え可）
- ページ別スクリーンショット例：
  - `home.png`：ホーム/ポータル
  - `login.png`：ログイン/登録
  - `admin-users.png`：管理者ユーザー管理
  - `competitions.png`：競技一覧
  - `profile.png`：プロフィール

命名ルールは snake_case/kebab-case のいずれかに統一してください。差分を小さく保つため画像の無駄な再圧縮は避けます。
